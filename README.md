# CollegeEduSystem 高校教务管理系统
JavaWeb课程设计分组项目，3人协作开发

## 小组分工
组员A(A-A-Free)：数据库设计、三层架构搭建、用户/课程/学期管理模块，分支admin-dev
组员B：前端页面、选课&成绩业务开发，分支teacher-student-dev
组员C：绩点计算、Excel导出、系统配置、文档部署，分支score-gpa-dev

## 分支使用规范
1. main：稳定主分支，存放可完整运行成品代码，禁止直接提交代码
2. admin-dev：组员A专属开发分支
3. teacher-student-dev：组员B专属开发分支
4. score-gpa-dev：组员C专属开发分支

## Git提交注释规范
格式：【模块】操作：修改内容
示例：
【数据库】完成全部数据表建表SQL
【架构】搭建SpringBoot三层基础项目
【管理员模块】实现课程新增、编辑接口
